Printing Your First Python Code

September 1, 2022 One-minute read
python • code • beginner

This article offers a very simple beginner help in printing your first python code, if one is new to programming.

Today I’ll teach y’all how to write a simple program that prints “Hello World” as its output.

This tutorial assumes you already have python installed on your computer. If not, search “python download” on Google Chrome and go to their website. Download for the type of device you have (in my case Windows PC) then click it once done downloading to install it.

With python installed, go to the file (either .txt or .py) and you can begin writing your code. This assumes you are using Visual Studio code as your editor as it’s my preferred one. If you haven’t installed it yet, here is a tutorial that will assist with that. Type the following on the editor and save the file:

print("Hello World")

If you used the “.txt” extension, you’ll perform the following steps in order to see the code you just wrote displayed.

  • STEP 1: Go to terminal and click on “New Terminal.” Your terminal will appear below your file.
  • STEP 2: Run the following command:
py -3 filename.txt

That’s all for today. Till next time guys, bye!
